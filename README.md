# David Lyons + RainFocus UI

## Dev Setup

Install dependencies:

```
npm install
```

Run the Vite development server:

```
npm run dev
```

## Build Instructions

Build the `src` files into the `build` folder:

```
npm run build
```

## Preview Instructions

Preview the production `build` folder at `http://localhost:4173`:

```
npm run preview
```

Or to preview without installing dependencies, run

```
cd build && npx http-server
```

then open `http://localhost:8080`
